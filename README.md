# Weather_Forecast
Weather forecasting program written in Python, that use recurrent neural networks to predict the weather on a given day and time using historical data. </br>

Network model use linear regression and recurrent neural network for compare them both.</br>

Data source: [Kaggle](https://www.kaggle.com/datasets/selfishgene/historical-hourly-weather-data)</br></br>


<strong>Projected data for selsected day and time:</strong></br>

<ul>
  <li><strong>Temperature</strong></li>
  <li><strong>Pressure</strong></li>
  <li><strong>Humidity</strong></li>
  <li><strong>Wind speed</strong></li>
  <li><strong>Wind direction</strong></li>
</ul>
